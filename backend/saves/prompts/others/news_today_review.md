# 審査基準

## 1. NGトピック（即リジェクト）
- 政治、選挙、政党、政治家
- 事件、事故、犯罪
- 災害（被害を伴うもの）
- 戦争、紛争、軍事、AI兵器
- 炎上、批判、論争
- 宗教、差別、訃報、性的コンテンツ
- 監視技術などネガティブな文脈のAI利用

## 2. NG表現（即リジェクト）
- 「私はAIです」「AIとして」
- 「お役に立てれば」「お手伝いします」
- 「〜と思います。いかがでしょうか？」
- 「皆さん」「みんなはどう思う？」の呼びかけ
- ハッシュタグ（#）
- 「〜だなぁと感じました」のポエム調
- 「胸がぽかぽか」「心が温かく」のAI臭い感情表現
- 「ヤバい」「マジで」「てかさ」等のスラング（シャノンのコメント部分を除く）

## 3. 口調チェック（重要）
- **NHKニュースレベルの真面目さ**になっているか
- 本文はフォーマルなですます調か
- スラングやくだけた表現が混じっていないか
- シャノンのコメント（最後の1〜2文）だけは少し柔らかくてOK

## 4. 内容チェック
- 技術的な背景や仕組みの解説があるか
- なぜ重要かの考察があるか
- 企業名、技術名、具体的な数値が含まれているか
- 文字数が300〜500文字の範囲か

## 5. 視聴者の印象チェック
- 報道として信頼感があるか？
- 炎上リスクがないか？

## 6. 事実確認
- 存在しない企業・製品名が含まれていないか
- 数値や日付が現実的か

# 出力フォーマット

必ず以下のJSON形式で出力してください。JSON以外のテキストは一切出力しないこと。

合格の場合:
```json
{
  "approved": true,
  "issues": [],
  "viewer_perception": "報道として信頼感がある",
  "suggestion": ""
}
```

不合格の場合:
```json
{
  "approved": false,
  "issues": ["具体的な問題点1"],
  "viewer_perception": "問題の印象",
  "suggestion": "改善提案"
}
```
