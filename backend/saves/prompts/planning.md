文脈を理解し、ユーザーに回答するために以下の json 形式で計画を立案してください

# 出力形式

```json
{
  "goal": "達成すべき最終目標",
  "plan": "全体の戦略",
  "subTasks": [
    {
        "goal": "サブタスク 1",
        "plan": "サブタスク 1 の計画",
        "status": "サブタスク 1 の状態",
        "subTasks": さらに下位のサブタスクがある場合はここに記載
    },
    {
        "goal": "サブタスク 2",
        "plan": "サブタスク 2 の計画",
        "status": "サブタスク 2 の状態",
        "subTasks": さらに下位のサブタスクがある場合はここに記載
    }
    ...
  ]
}
```

status は以下のいずれかを使用してください

- pending
- in_progress
- completed
- error
