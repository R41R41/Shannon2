### 2本目（技術ハイライト）

```
一番ハマったのはこれ。

twitterapi.io の create_tweet_v2 に
attachment_url を渡しても引用RTにならない問題。

解決策：URLをツイートテキストの末尾に付加するだけ

// ❌ これは普通のツイートになる
{ tweet_text: "コメント", attachment_url: "..." }

// ✅ これで引用RTになる
{ tweet_text: "コメント https://x.com/..." }

ドキュメントに書いてないので1時間溶かした
```

**ポイント**: エンジニアが「それ知りたかった！」となる具体的なtips。コードスニペットは視覚的に目立つ。
